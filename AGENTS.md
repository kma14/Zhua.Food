# Codex Project Guide

Codex should use `CLAUDE.md` as the shared agent rules file for this repository.

Read `CLAUDE.md` before doing non-trivial work. It contains the distilled hard rules, commands, gotchas, and conventions used by both Claude Code and Codex.

`plan-cc.md` remains the authoritative planning, architecture, milestone, and decision-log document. If architecture, roadmap, or decisions change, update `plan-cc.md`; if working rules change, update `CLAUDE.md`.
